# stratum — 機能要件

## 決定済み

### 投稿

- Markdown 形式で記述
- 必須項目は**本文のみ**
- タイトルは任意（未入力時は本文冒頭を自動採用）
- タグは任意設定
- 公開範囲は任意設定

### 公開範囲

| 範囲 | 説明 |
|------|------|
| private | 自分のみ閲覧可能 |
| グループ | 指定したグループのメンバーが閲覧可能 |
| 複数グループ | 複数のグループを指定可能 |
| public | 組織全体に公開 |

- デフォルトは **public**
- デフォルト値は組織単位・個人単位で変更可能

### 外部公開リンク

記事単位で外部公開リンクを発行できる機能。

- 認証不要の URL を発行
- 記事単位で ON/OFF を切り替え可能
- 有効期限の設定
  - プリセット: 1日 / 7日 / 30日 / 無期限
  - カスタム日時指定も可能
- 手動で無効化可能
- 投稿者退会時に自動無効化
- 組織管理者による制御
  - 外部公開リンク機能自体の ON/OFF
  - 有効期限の上限設定

### 検索

- 自然言語による意味検索（ON/OFF 切替可能）
- タグ、投稿者などによる基本絞り込み

### タイムライン

全体タイムラインとカスタムフィードのハイブリッド構成。

- **全体タイムライン**: 閲覧権限のある投稿が時系列で表示される
- **カスタムフィード**: タグ / グループ / メンバーで絞り込んだフィードを自分で作成可能

### ダッシュボード

タイル型ウィジェットを自由に配置できるカスタマイズ可能なダッシュボード。

- ウィジェットのサイズは可変（縦長・短いなど）
- ウィジェット種類
  - フィード（カスタムフィードを表示）
  - ピン留め（重要な投稿を固定表示）
  - 最近の検索・閲覧履歴
  - 自分の下書き

### 退会時の扱い

- private 以外の投稿は組織内で継続閲覧可能
- 外部公開リンクは自動無効化
